Policy

If you like to submit your website in our search engine then we are also like to provide your website in the search result of our ePavitram search engine. We provide general guide lines for submission of website.
Website must contain the following meta information so that our ePavitram web crawler can easily crawl the website for perpose of indexing and store into database.
1. Every web page should contain title tag. Which must contain Title of web page.
2. Every web page should contain meta tag which must contain the Description of web page.
3. Every web page should contain meta tag which must contain Keywords of webpage.
4. Your website must have sitemap so that ePavitram web crawler crawl all pages whose URL are provided into sitemap.
5. Your website will be available in the ePavitram search result whithin only one week. After you have submitted your website.
6. If you have any changed in meta information about your website then you must submit url or sitemap of that web pages.
7.Type of website should be multi page application website.
